# Module Boundaries

## core_platform
- health checks
- backup records and archive creation
- audit logging
- app settings helpers

## organization
- company profile
- branch registry
- fiscal period foundation
- document sequence foundation

## identity
- login/logout/me
- user directory
- self-account flows
- roles, permissions, assignments

## Out of scope for first implementation
- customers
- services catalog
- dresses
- bookings
- payments workflows beyond backup/history foundation
- accounting UI and posting engine
